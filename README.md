# Devil-Guard

This repository contains the full Devil-Guard project split into three top-level folders:

- Devil-Guard-PC: Windows desktop and service solution (client, setup, overseer, sentinel, patcher, core libraries).
- Devil-guard-server: Standalone Gatekeeper server project.
- Devil-Guard-Web-Upload: PHP web portal and API for administration, auth, and attestation integration.

## Folder Guide

### Devil-Guard-PC
Main .NET solution and desktop/service components.

### Devil-guard-server
Minimal .NET Gatekeeper service project used for attestation endpoints.

### Devil-Guard-Web-Upload
Web upload package with PHP source, SQL schema, admin portal, and API routes.
